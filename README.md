# Portfolio Website

This is a simple and responsive portfolio website built using HTML, CSS, and JavaScript.

The project is designed to be easy to read, edit, and customize for beginners and developers.

## How to Edit
- Update text and sections in the HTML files
- Replace images to change visuals
- Edit CSS files to change colors, layout, or design
- Edit JavaScript files for interactive features

## Features
- Responsive design
- Clean and simple UI
- Easy to customize
- Beginner friendly structure

## Technologies
- HTML
- CSS
- JavaScript
